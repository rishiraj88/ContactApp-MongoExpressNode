# Contact App (Mongo) [Draft Version]
The contact wallet, hosted in Docker container.


## Functionalities:
- Save email ID and name of a person.
- Retrieve the person's details.
- Update the person's details.

## Built with Tech Stack:

### Frontend-
- HTML
- CSS
- JavaScript

### Backend-
- Node.js
- Mongo Express

### Database-
- MongoDB
